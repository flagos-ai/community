# sig-kernelgen: Kernel Code Generation

## Scope

**Responsible Modules:**
- [KernelGen](https://github.com/flagos-ai/KernelGen) — AI Kernel Code Generation
- [KernelGenBench](https://github.com/flagos-ai/KernelGenBench) — Kernel Generation Evaluation

**In Scope:**
- Kernel code generation engine development
- Kernel template library maintenance
- Multi-chip Kernel generation support
- Kernel correctness and performance verification

**Out of Scope:**
- Hand-written operator optimization → sig-operator
- Compiler IR → sig-compiler
