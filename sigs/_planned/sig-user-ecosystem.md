# sig-user-ecosystem: User Ecosystem (Planned)

> The community needs a growth engine: user story propagation, developer onboarding, community events, and external ecosystem integration.

## Reason for Deferred Activation

The current user base is small; a standalone community operations team is not yet needed. Activation will occur when GitHub Discussions monthly active users reach ≥50 and external contributors reach ≥10.

## Planned Responsibilities

- Collect and promote user stories
- Developer onboarding (mentorship, good first issues)
- Community events (online meetups, hackathons, annual conferences)
- Social media operations
- External ecosystem integration (Hugging Face, PyTorch, vLLM)

## Current Interim Arrangement

The TSC Community Manager serves in this capacity, starting with a community newsletter and a good first issue label system.
