# Incubating Project Records

One directory per project, holding its full process records:

```
projects/<project-name>/
├── proposal.md      # Donation proposal (from ../proposal-template.md),
│                    #   with Final Acceptance and execution records appended
├── ip-checklist.md  # Completed and verified IP clearance checklist copy
└── reviews/         # Archived annual reviews (per ../annual-review-template.md)
```
